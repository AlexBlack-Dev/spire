# Улучшение качества иконок для Windows

## Проблема
Иконки в Windows Taskbar отображаются в плохом разрешении из-за:
1. Отсутствия иконок высокого разрешения (256x256)
2. Неполного набора размеров в ICO файле
3. Низкого качества существующих PNG файлов

## Решение

### Шаг 1: Создание высококачественных иконок

#### Вариант A: Использовать онлайн-конвертер (рекомендуется)
1. Перейдите на https://convertio.co/svg-ico/
2. Загрузите файл `blum-avatar.svg`
3. Выберите следующие размеры:
   - 16x16
   - 32x32  
   - 48x48
   - 64x64
   - 128x128
   - 256x256
4. Скачайте полученный ICO файл

#### Вариант B: Использовать ImageMagick (если установлен)
```bash
# Установите ImageMagick с https://imagemagick.org/script/download.php
# Или через Chocolatey: choco install imagemagick

# Создайте PNG файлы разных размеров
magick convert -background transparent -resize 16x16 "blum-avatar.svg" "16x16.png"
magick convert -background transparent -resize 32x32 "blum-avatar.svg" "32x32.png"
magick convert -background transparent -resize 48x48 "blum-avatar.svg" "48x48.png"
magick convert -background transparent -resize 64x64 "blum-avatar.svg" "64x64.png"
magick convert -background transparent -resize 128x128 "blum-avatar.svg" "128x128.png"
magick convert -background transparent -resize 256x256 "blum-avatar.svg" "256x256.png"

# Создайте ICO файл со всеми размерами
magick convert 16x16.png 32x32.png 48x48.png 64x64.png 128x128.png 256x256.png "icon.ico"
```

### Шаг 2: Замена файлов
1. Замените `src-tauri/icons/icon.ico` на новый файл
2. При необходимости обновите PNG файлы в `src-tauri/icons/`:
   - `16x16.png`
   - `32x32.png`
   - `48x48.png`
   - `64x64.png` (создать новый)
   - `128x128.png`
   - `256x256.png` (создать новый)

### Шаг 3: Пересборка приложения
```bash
npm run tauri build
```

### Шаг 4: Установка обновленного приложения
1. Удалите старую версию из Windows
2. Установите новую версию из:
   - `src-tauri/target/release/bundle/msi/BLUM_1.0.0_x64_en-US.msi`
   - Или `src-tauri/target/release/bundle/nsis/BLUM_1.0.0_x64-setup.exe`

## Почему это важно для Windows
Windows использует разные размеры иконок для разных контекстов:
- **16x16**: Мелкие значки в проводнике
- **32x32**: Средние значки
- **48x48**: Крупные значки
- **64x64**: Увеличенные значки для HiDPI
- **128x128**: Очень крупные значки
- **256x256**: Максимальное качество для 4K/Retina дисплеев

Windows Taskbar обычно использует 32x32 или 48x48, но для HiDPI дисплеев может потребоваться 64x64 или 128x128.

## Альтернативное решение
Если качество иконок все еще неудовлетворительное, можно:
1. Создать упрощенную версию иконки для малых размеров
2. Использовать более контрастные цвета для лучшей читаемости
3. Увеличить толщину линий в векторной графике для малых размеров

## Проверка результата
После обновления иконок:
1. Установите приложение
2. Проверьте иконку в панели задач Windows
3. Проверьте иконку в меню "Пуск"
4. Проверьте иконку на рабочем столе
5. Проверьте иконку в проводнике файлов
